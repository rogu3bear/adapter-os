# Tools

Development and operational tooling.

## Contents

- [`inventory/`](inventory/) - Tool inventory and registry

## Purpose

This directory contains auxiliary tools and utilities for:

- Development workflows
- Operational maintenance
- System diagnostics
- Tool inventory management

## Organization

Tools are organized by functional area. The `inventory/` subdirectory maintains a registry of all available tools and their purposes.

## Related

- [`scripts/`](../scripts/) - Executable automation scripts
- [`xtask/`](../xtask/) - Cargo xtask build utilities

## Usage

Tools in this directory are typically invoked directly or referenced by automation scripts. Check individual tool documentation for specific usage.
