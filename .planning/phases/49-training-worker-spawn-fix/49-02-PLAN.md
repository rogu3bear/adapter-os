<objective>
Harden the training worker supervisor with circuit breaker crash protection and in-flight job cleanup on worker crash.

Purpose: The current supervisor retries indefinitely with backoff but never stops, wasting resources on a crash loop. When the worker crashes, running training jobs are left in "running" state forever. This plan adds a sliding-window circuit breaker (3 crashes in 5 minutes = permanently degraded) and marks in-flight jobs as failed with an actionable reason.

Output: `WorkerCircuitBreaker` struct in the supervisor, `mark_running_jobs_failed_worker_crash()` in DB, degraded marker lifecycle management.

```rust
use std::collections::VecDeque;

struct WorkerCircuitBreaker {
    crash_timestamps: VecDeque<tokio::time::Instant>,
    max_crashes: u32,
    window: Duration,
    tripped: bool,
}

impl WorkerCircuitBreaker {
    fn new(max_crashes: u32, window: Duration) -> Self {
        Self {
            crash_timestamps: VecDeque::new(),
            max_crashes,
            window,
            tripped: false,
        }
    }

    /// Record a crash and return whether the circuit breaker has tripped.
    fn record_crash(&mut self) -> bool {
        let now = tokio::time::Instant::now();
        self.crash_timestamps.push_back(now);
        // Evict crashes outside the sliding window
        while let Some(&front) = self.crash_timestamps.front() {
            if now.duration_since(front) > self.window {
                self.crash_timestamps.pop_front();
            } else {
                break;
            }
        }
        if self.crash_timestamps.len() >= self.max_crashes as usize {
            self.tripped = true;
        }
        self.tripped
    }

    fn is_tripped(&self) -> bool {
        self.tripped
    }
}
```
