  <files>

  <action>
Create `bootstrap.sh` at project root. Make it executable (`chmod +x`).

Structure:
1. **Shebang and strict mode:** `#!/usr/bin/env bash` with `set -euo pipefail`.

2. **SCRIPT_DIR detection:** Resolve script's own directory for relative paths.

3. **Prerequisite checks (fail-fast):**
   - Check `brew` command exists. If missing: print "Homebrew is required. Install from https://brew.sh" and exit 1.
   - Check `rustup` command exists. If missing: print "Rust toolchain (rustup) is required. Install from https://rustup.rs" and exit 1.
   - Check `cargo` command exists (should come with rustup). If missing: print "cargo not found. Run: rustup install stable" and exit 1.

4. **Idempotent dependency installation:**
   Use a helper function `check_brew_pkg()` that checks `brew ls --versions "$pkg" >/dev/null 2>&1` before installing. Print `[ok] pkg` if present, `[installing] pkg...` if not.

   Install these brew packages (sequential to avoid lock contention):
   - `ml-explore/mlx/mlx` (MLX headers for FFI)

   Check and install these rustup components:
   - WASM target: `rustup target list --installed | grep -q wasm32-unknown-unknown || rustup target add wasm32-unknown-unknown`

   Check for `wasm-bindgen-cli`:
   - `command -v wasm-bindgen >/dev/null 2>&1 || cargo install wasm-bindgen-cli`

   Check for `trunk` (WASM build tool):
   - `command -v trunk >/dev/null 2>&1 || cargo install trunk`

5. **Summary:** Print all deps checked/installed with status.

6. **--verify flag handling:**
   Parse args with simple `case` pattern. If `--verify` is passed:
   - Run `cargo check --workspace` and report pass/fail
   - Run `./scripts/ui-check.sh` and report pass/fail
   - Exit with appropriate code
   If no `--verify`, print "Dependencies ready. Run with --verify to validate the full build." and exit 0.

7. **--help flag:** Print usage showing both modes.

Do NOT use complex argument parsing -- just a simple `for arg in "$@"` or `case "$1"` pattern. The script has exactly two optional flags: `--verify` and `--help`.

Total script should be ~80-120 lines. Keep it readable and well-commented.
  </action>
