# Requirements: adapterOS

**Defined:** 2026-02-23
**Core Value:** Deterministic, verifiable LoRA inference on Apple Silicon -- every response is reproducible and cryptographically attested

## v1 Requirements

Requirements for MVP production readiness. Each maps to roadmap phases.

### Compilation

- [x] **COMP-01**: Full workspace compiles cleanly with `cargo check`
- [x] **COMP-02**: All deleted DB module consumers (`query/`, `util/`) updated to use replacement APIs
- [x] **COMP-03**: All CI gate workflows pass (ci, stability, determinism, integration, security, migration)
- [x] **COMP-04**: Foundation run script (`scripts/foundation-run.sh`) passes end-to-end (build, boot, smoke)
- [x] **COMP-05**: P0 dependency upgrades applied (sqlx 0.8.2->0.8.6, safetensors 0.4->0.7.0, tokio 1.35->1.44)
- [x] **COMP-06**: SQLx offline cache synchronized with current migrations

### FFI Safety

- [ ] **FFI-01**: All 187 unsafe blocks in `adapteros-lora-mlx-ffi` audited with SAFETY comments
- [ ] **FFI-02**: unwrap/expect calls in non-test FFI code replaced with Result propagation
- [ ] **FFI-03**: AddressSanitizer (ASAN) enabled in CI for FFI crate tests
- [ ] **FFI-04**: Concurrent adapter hot-swap tested under load without memory corruption

### Determinism

- [ ] **DET-01**: Q15 canonicalization extended to all values entering receipt hashes
- [ ] **DET-02**: All unquantized layers in inference pipeline mapped and determinism envelope documented
- [ ] **DET-03**: Runtime MLX version check at boot (compiled vs. installed version match)
- [ ] **DET-04**: Replay harness passes 100% on determinism test suite
- [ ] **DET-05**: Fast-math flag scan passes (no forbidden compiler flags in build)

### OpenAI API

- [ ] **API-01**: Structured output via `response_format` parameter (JSON mode with schema enforcement)
- [ ] **API-02**: Function/tool calling via `tools` and `tool_choice` parameters
- [ ] **API-03**: Complete token usage reporting in all completion responses (`prompt_tokens`, `completion_tokens`, `total_tokens`)
- [ ] **API-04**: Missing OpenAI parameters supported (`seed`, `stop`, `frequency_penalty`, `presence_penalty`, `logprobs`)
- [ ] **API-05**: OpenAPI spec in sync with code (contract drift resolved)
- [ ] **API-06**: Error responses follow OpenAI error format (`error.type`, `error.code`, `error.message`)

### Observability

- [ ] **OBS-01**: Prometheus metrics exported for inference latency (TTFT, TPS histograms), adapter load times, memory pressure
- [ ] **OBS-02**: OpenTelemetry trace propagation end-to-end (request -> control plane -> worker -> kernel -> response)
- [ ] **OBS-03**: Graceful shutdown completes in-flight requests when receiving SIGTERM during streaming inference
- [ ] **OBS-04**: mimalloc integration as system allocator (replaces default for long-running service stability)
- [ ] **OBS-05**: Memory budget enforcement tracking MLX unified memory + Rust heap

### Production Operations

## Out of Scope

| Feature | Reason |
|---------|--------|
| GPU/CUDA backend | Apple Silicon only -- MLX/Metal/CoreML is the platform |
| Python runtime or PyO3 | Pure C FFI to MLX -- no Python dependency, no startup overhead |
| Kubernetes orchestrator | Single-node deployment -- K8s adds complexity without benefit |
| Custom model format | safetensors is the standard; AOS wraps it, doesn't replace it |
| Framework-specific SDKs | Full OpenAI API compatibility is the universal integration point |
| Mobile app | Web-first -- Leptos CSR covers all platforms |
| Multi-node clustering | Single-node focus for MVP -- federation is communication, not clustering |
| Tailwind CSS | Liquid Glass design system -- pure CSS |
