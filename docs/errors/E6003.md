# E6003 - Router Skew Detected

## Description
Router gate distribution exceeds entropy floor.

## Cause
The router is not distributing gates evenly across adapters, indicating:
- Router calibration issues
- Adapter activation patterns are skewed
- Entropy floor setting too low
- Adapter quality disparities

## Fix
1. Check router calibration
2. Verify entropy_floor setting
3. Review adapter activation patterns
4. Rebuild Plan if needed
