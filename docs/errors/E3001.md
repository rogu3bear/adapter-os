# E3001 - Kernel Manifest Signature Invalid

## Description
The Metal kernel manifest signature verification failed.

## Cause
The kernel manifest signature doesn't match the expected signature, indicating:
- Kernels were compiled with different flags
- The manifest was modified after signing
- The signing key doesn't match the runtime key

## Fix
1. Rebuild kernels: `cd metal && ./build.sh`
2. Verify CI signing key
3. Check kernel hash: `aosctl verify-kernel`
4. Ensure toolchain.toml matches build environment
