# E4002 - Telemetry Write Failed

## Description
Cannot write telemetry events to bundle.

## Cause
The telemetry system cannot write events to the bundle, typically due to:
- Insufficient disk space
- Permission issues
- Bundle size limits exceeded
- File system corruption

## Fix
1. Check disk space: `df -h var/telemetry`
2. Verify write permissions
3. Check bundle size limits in policy
4. Review telemetry configuration

## Examples
```bash
# Check disk space
df -h var/telemetry

# Check permissions
ls -la var/telemetry/

# Verify telemetry configuration
cat configs/cp.toml | grep telemetry
```
