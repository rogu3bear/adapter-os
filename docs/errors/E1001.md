# E1001 - Invalid Signature

## Description
The Ed25519 signature verification failed for an artifact or bundle.

## Cause
The signature in the bundle does not match the embedded public key, indicating either:
- The bundle has been tampered with
- The wrong signing key was used
- The signature was corrupted during transfer

## Fix
1. Verify the public key is correct
2. Check that the bundle hasn't been modified
3. Re-sign the bundle: `aosctl sign-bundle <bundle>`
4. Verify signature: `aosctl verify <bundle>`

```bash
# Verify bundle signature
aosctl verify artifacts/adapters.zip

# Re-sign bundle
aosctl sign-bundle artifacts/adapters.zip

# Check public key
cat artifacts/adapters.zip/public_key.hex
```
