This directory contains deprecated files that have been moved from the top-level directory due to being stale, outdated, or containing inaccurate information.

- `BEST_PRACTICE_VIOLATIONS_PATCH_PLAN.md` - Claims completion but references non-existent crates

- `CONSOLIDATION_PATCH_PLAN.md` - Outdated consolidation plan
- `CORRECTIVE_PATCH_PLAN.md` - Claims fixes for non-existent issues

- `PATCH_COMPLETION_PLAN.md` - References non-existent phases

- `ui-improvement-patch-plan.md` - Outdated UI improvement plan

### Status Reports (Inaccurate)
- `EXECUTION_SUMMARY.md` - Claims execution of non-existent phases

### Temporary Files
- `fix_multiline_error_response.py` - Temporary Python script
- `grafana.json` - Outdated Grafana configuration
- `registry.db` - Old registry database file
- `requirements-mlx.txt` - Outdated Python requirements

## Archive Policy

- Files are moved here when they contain outdated information
- No active maintenance is performed on deprecated files
- Files may be removed after a grace period if no longer needed for reference
- Links to deprecated files should be updated to point to current equivalents
