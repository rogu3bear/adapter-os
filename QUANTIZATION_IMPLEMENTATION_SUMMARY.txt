================================================================================
KEY FEATURES
================================================================================

QUANTIZATION METHODS:
- INT8: 8-bit signed quantization with per-group symmetric scaling
  * Compression: 4x (float32 → int8)
  * Accuracy: ~99% preserved
  * Use: Production inference

- INT4: 4-bit quantization with byte-packing
  * Compression: 8x (float32 → 4-bit packed)
  * Accuracy: ~95% preserved
  * Use: Memory-constrained environments

QUANTIZATION METRICS:
- Mean Squared Error (MSE) - Per-element quantization error
- Signal-to-Noise Ratio (SNR) - Quality in dB
- Maximum Absolute Error - Outlier detection
- Compression Ratio - Size reduction verification

SAFETENSORS SUPPORT:
- Parse safetensors file headers efficiently
- Extract tensor metadata (dtype, shape, offset)
- Load tensors as float32 or quantized formats
- Support for multiple datatypes in single file

API SURFACE
================================================================================

PUBLIC EXPORTS (from lib.rs):
  pub use quantization::{
    MLXQuantizer, QuantizationConfig, QuantizationMetadata,
    QuantizationStats, QuantizedTensor, WeightCompressor,
  };
  pub use safetensors_loader::{SafetensorsLoader, TensorInfo};

KEY FUNCTIONS:
  Quantization:
    - MLXQuantizer::quantize_int8(&data, group_size, &shape) → QuantizedTensor
    - MLXQuantizer::quantize_int4(&data, group_size, &shape) → QuantizedTensor
    - MLXQuantizer::dequantize_int8(&tensor) → Vec<f32>
    - MLXQuantizer::dequantize_int4(&tensor) → Vec<f32>
    - MLXQuantizer::calculate_stats(&original, &quantized) → QuantizationStats

  Loading:
    - SafetensorsLoader::load(path) → SafetensorsLoader
    - loader.load_tensor_as_f32(name) → (Vec<f32>, Vec<i32>)
    - loader.tensor_names() → Vec<String>
    - loader.get_tensor_info(name) → Option<&TensorInfo>
