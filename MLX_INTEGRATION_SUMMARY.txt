BUILD COMMANDS
================================================================================

# Stub build (no MLX C++ required)
cargo build -p adapteros-lora-mlx-ffi

# Real MLX build (requires MLX C++ library)
cargo build -p adapteros-lora-mlx-ffi --features real-mlx

# Full workspace with MLX enabled
cargo build --workspace --features multi-backend,real-mlx --release

================================================================================
DEPLOYMENT QUICK START
================================================================================

1. Install MLX Library:
   brew install mlx

2. Build with MLX:
   cargo build -p adapteros-lora-mlx-ffi --features real-mlx --release

3. Prepare Model:
   python -m mlx_lm.convert --hf-path Qwen/Qwen2.5-7B-Instruct --mlx-path models/

4. Start Server:
   export AOS_MLX_FFI_MODEL="./models/qwen2.5-7b-mlx"
   ./target/release/aosctl serve --backend mlx --model-path ./models/qwen2.5-7b-mlx

5. Test Inference:
   curl -X POST http://localhost:8080/v1/infer \
     -H "Content-Type: application/json" \
     -d '{"prompt": "Hello", "max_tokens": 10}'
